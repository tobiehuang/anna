# 塔羅解牌練習室 — 上線部署教學（Cloudflare Workers）

照著做大約 20～30 分鐘就能上線。全程免費（Cloudflare 免費方案每天可承受 10 萬次請求），你只需要付 Claude API 的使用費。不需要會寫程式，只需要複製貼上。

## 事前準備

需要兩樣東西：一個 Anthropic 帳號的 API 金鑰（在 platform.claude.com 建立，sk-ant- 開頭，並先儲值，建議從 25 美元開始），以及 `worker.js` 這個檔案（我已經做好給你了）。

## 步驟一：註冊 Cloudflare

到 https://dash.cloudflare.com/sign-up 用 email 註冊一個免費帳號，過程不需要信用卡。

## 步驟二：建立 Worker

1. 登入後，左邊選單找到「Compute (Workers)」或「Workers & Pages」，點進去。
2. 點「Create（建立）」→ 選「Start with Hello World」（或類似的空白範本）→ 幫它取個名字，例如 `tarot-practice`，這個名字會變成網址的一部分 → 點「Deploy（部署）」。
3. 部署完成後，點「Edit code（編輯程式碼）」進入程式編輯器。
4. 把編輯器裡原有的程式**全部刪掉**，打開我給你的 `worker.js`，全選、複製、貼上。
5. 點右上角「Deploy（部署）」儲存。

## 步驟三：設定金鑰和課程密碼（最重要的一步）

1. 回到這個 Worker 的頁面，進入「Settings（設定）」→ 找到「Variables and Secrets（變數和機密）」。
2. 新增第一個機密：點「Add」，Type 選 **Secret**，名稱填 `ANTHROPIC_API_KEY`（大小寫要一模一樣），值貼上你的 sk-ant- 金鑰，儲存。
3. 新增第二個機密：同樣選 **Secret**，名稱填 `COURSE_PASSWORD`，值填你要發給學員的課程密碼（自己取，例如 `tarot2026`），儲存。
4. 兩個都要選 Secret 類型，這樣任何人（包括你自己）之後都看不到內容，只能覆寫。

## 步驟四：測試

Worker 頁面上會顯示你的網址，長得像 `https://tarot-practice.你的帳號.workers.dev`。用瀏覽器打開，輸入課程密碼，填一個牌局測試對話。手機也可以直接用。

## 步驟五：發給學員

給學員兩樣東西就好：網址和課程密碼。提醒學員不要把密碼外流。如果密碼流出去了，回到步驟三把 `COURSE_PASSWORD` 改成新的值，舊密碼立刻失效（學員的瀏覽器會自動跳出重新輸入密碼的視窗）。

## 費用與安全控管

到 platform.claude.com 的帳單設定裡設定「每月用量上限」，例如 50 美元，超過就自動停止，絕對不會爆卡。一場完整的解牌練習成本大約台幣 1～3 元；假設 30 位學員每人每週練 3 場，一個月大約是台幣 400～1,100 元的 API 費用。Cloudflare 端則完全免費。

程式內建了幾道保險：單場對話最多 60 則訊息（避免無限聊天燒額度）、單則訊息最長 4000 字、密碼錯誤直接擋下不會呼叫 API。

## 之後想改東西

- **改老師的個性、牌義系統**：編輯 `worker.js` 最上面的 SYSTEM_PROMPT 段落，改完在 Cloudflare 編輯器貼上、重新 Deploy 即可，學員什麼都不用做。
- **加入你的教材**：把教材文字交給 Claude 整合進 SYSTEM_PROMPT，再重新部署。
- **換課程密碼**（例如每一期課換一次）：改 `COURSE_PASSWORD` 這個 Secret 就好，不用動程式。
- **想要自己的網域**（例如 tarot.你的品牌.com）：Cloudflare Worker 設定裡的「Domains & Routes」可以綁自訂網域，需要時再回來問 Claude。

## 常見問題

**學員說「課程密碼不正確」但密碼是對的？** 檢查步驟三的 Secret 名稱是否完全是 `COURSE_PASSWORD`，以及值的前後有沒有多打空格。

**畫面顯示「老師的 API 金鑰失效了」？** 到 platform.claude.com 檢查金鑰是否被刪除、或帳戶餘額是否用完。

**想看有多少人在用？** Cloudflare Worker 頁面有免費的流量分析（Metrics），可以看到每天的請求次數。
